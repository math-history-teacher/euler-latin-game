# Euler's Latin Reading Game
### オイラー・ラテン語読解ゲーム

このプロジェクトは、レオンハルト・オイラーの1735年の論文『自然数の累乗から生じる逆数の級数の和に関する第二の論文（De summis serierum reciprocarum...）』の第1節を題材にした、ラテン語学習ゲームです。

## 収録内容
* **学習モード**: 論文の原文、和訳、重要語彙、文法解説の閲覧
* **単語クエスト**: 論文に登場する数学・ラテン語の4択クイズ
* **文法クエスト**: 構文や格変化に関するクイズ
* **和訳クエスト**: チャンクを並べ替えて正しい日本語訳を作るパズル

## 使い方
1. このリポジトリの `index.html` をブラウザで開くか、GitHub Pagesで公開されたURLにアクセスしてください。
2. モバイル・デスクトップ両方の画面サイズに対応しています。

## 出典
Euler, L. (1735). *De summis serierum reciprocarum ex potestatibus numerorum naturalium ortarum dissertatio altera*.
